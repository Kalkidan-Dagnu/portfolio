A collaborative question-and-answer site created as part of the Evangadi Bootcamp. Developed in a team using React for an engaging front end, Node.js and Express for server management, and MySQL for secure, structured data storage.

Amazon

A feature-rich e-commerce platform designed to practice advanced skills in authentication, authorization, and payment integration. Built with React for a responsive UI, Node.js and Express for backend handling, Firestore as a scalable database, and Firebase authentication with email/password sign-in. Netflix

A movie browsing site that leverages The Movie Database (TMDB) API to display popular films and shows. Created to practice API integration techniques and enhance front-end development skills with React.

Apple HomePage

A visually engaging replica of the Apple website, built to strengthen skills in responsive web development. Uses React for dynamic front-end rendering, Node.js for server-side handling, and Bootstrap for a modern, mobile-friendly layout.
